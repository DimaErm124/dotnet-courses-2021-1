Данная программа предназначена для создания бинарного дерева поиска.
Основным классом является обобщенный абстрактный класс, передаваемый тип данных (T) которого должен реализовывать интерфейс IComparable<T>.
От него наследуются два класса, в которых алгоритмы добавления, поиска, перебора реализованы рекурсивно и итерационно.

Также имеются тесты по следующим типам данных:
- int;
- string;
- ExamResult (специально созданный класс для моделирования дерева с результатами экзаменов студентов).
