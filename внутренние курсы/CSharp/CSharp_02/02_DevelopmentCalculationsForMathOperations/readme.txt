Данное решение состоит из двух частей:
1) Первая часть программы предназначена для вычисления кореня n-ой степени из числа:
	- методом Ньютона с заданной точностью;
	- системными функциями.
Сравнить результаты обоих способов.

2) Вторая часть программы предназначена для конвертации неотрицательного целого числа в строку, содержащую:
	- двоичное представление числа;
	- восьмиричное представление числа;
	- шестнадцатиричное представление числа;
	- представление числа в виде Base32;
	- представление числа в виде Base64.
Реализовать конвертацию системными функциями, а также разработать свои классы и методы.
Сравнить результаты обоих способов.
